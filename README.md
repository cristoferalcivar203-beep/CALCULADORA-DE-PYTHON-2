# Calculadora en Python

Esta es una calculadora creada con Python y Streamlit.

## Funciones
- Suma
- Resta
- Multiplicación
- División

## Cómo usar la calculadora

1. Ingresar el primer número.
2. Ingresar el segundo número.
3. Seleccionar la operación matemática.
4. Presionar el botón "Calcular".
5. Ver el resultado en pantalla.

## Tecnologías usadas
- Python
- Streamlit
- GitHub
